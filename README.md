
# meteorbox

Meteorbox is a tool that aids the building and deployment of meteorapps. It includes scaffolding, checking, and environment validation.

## Features

- simple single file
- pure bash script

## To use

a cli menu is built into meteorbox, to run

$ sh mb.sh
